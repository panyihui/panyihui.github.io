C语言中const以及其与define的区别

By Pan

2018-01-23 更新日期:2018-01-23

文章目錄

·const是用来限定一个变量不允许被改变的修饰符。被const修饰的词是只读变量。

·const与define的区别
1）const是关键字，define不是。
2）const可以进行类型检查，define不可以。因为宏定义常量没有数据类型。
3）举个栗子：

#define AB 4+5
则表达式2*AB*3就相当于是2*4+5*3，结果是8+15=23   （C语言考试就老爱考define宏定义的用法，一不留神就上当了）

const int AB = 4+5
则表达式2*AB*3就相当于是2*9*3，结果是54

·值得注意的是：用一个const变量来初始化数组，编译器会报错。
比如：
const int n = 3;
double a[n];
因为在C语言中，数组的大小应该是个常量，而const int n中的n只是一个变量。

·“尽量使用编译器而不用预处理”
我的C语言老师上课的时候，const的部分是一笔带过的，反而define讲的多、用的也多，这就导致我一开始写编程作业几乎没用到const。
后来一直没弄明白const和define的差别，就搜了很多大佬的回答和心得，印象最深的就是这句话“尽量使用编译器而不用预处理”。http://m.jb51.net/article/34163.htm
